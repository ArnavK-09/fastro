# Publishing and Deployment

- Install and init [gcloud sdk](https://cloud.google.com/sdk). See [the quickstart.](https://cloud.google.com/sdk/docs/quickstart)

- Init a project
    ```
    fastro init
    ```
- Deploy and publish your app
    ```
    fastro deploy
    ```
